\chapter{Diff Listings}
\label{app:diffs}

This appendix contains the full listing of the differences obtain when running the \texttt{diff} utility on the trace generated through Vivado's simulator (on the left, lines starting with \texttt{>}) against the trace generated with our MLIR-based simulator (on the right, lines starting with \texttt{<}). Note that a listing for the graycode example is not presents, as no diff is generated for that example. Also, due to the large amount of variable don't care values (\texttt{x}) present in the Snitch example, we only list the diff for the first $50$ns for that example.
